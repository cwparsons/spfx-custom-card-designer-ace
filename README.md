# spfx-card-designer-ace

## Summary

A generic adaptive card extension, inspired by the *Card designer* extension.
Allows for creating dynamic properties in the property pane, instead of using
JSON for the quick view data.

![version](https://img.shields.io/badge/version-1.13.1-green.svg)

## Applies to

- [SharePoint Framework](https://aka.ms/spfx)

## Minimal Path to Awesome

- Clone this repository
- Ensure that you are at the solution folder
- in the command-line run:
  - **npm install**
  - **npm run serve**
